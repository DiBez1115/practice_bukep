git clone - создаёт копию репозитория на ПК пользователя
git add - добавление содержимого файла в индекс
git mv - переместить или переименовать файл, каталог или символическую ссылку
git diff - показать изменения между коммитами
git status - показывает состояние рабочего дерева
git commit - запись изменений в репозиторий
git pull - извлечь из другого репозитория или локальной ветки и интегрировать с ней
git push - обновить удалённые данные, используя локальные
.gitignore нужен для скрытия файлов и папок от системы Git. Обычно туда входят ненужные файлы.
