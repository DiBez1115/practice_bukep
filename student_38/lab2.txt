Для выполнения задания использовались команды:
git clone https://github.com/DiBez1115/practice_bukep.git (Для клонирования удаленного репозитория.)
cd practice_bukep (Для перехода в папку локального репозитория.)
git branch evoshny (Для создания ветки evoshny.)
git branch (Для просмотра существующих веток.)
git checkout evoshny (Для перехода на ветку evoshny.)
git add -A (Для внесения изменений на ветке evoshny.)
git commit -m "Lab2" (Для сохраняет изменения индексов в историю на ветке evoshny.)
git checkout main (Для перехода на ветку main.)
git merge (Находясь на ветке main производится слияние с веткой evoshny.)
git add -A (Для внесения изменений в индекс на ветке main.)
git commit -m "Lab2" (Сохраняет изменения индексов в историю на ветке main.)
git pull (Для сравнения и внесения изменений в локальный репозиторий если они необходимы на ветке main.)
git push (Для синхронизации локального и удалённого репозитария на ветке main.)
Файл .gitignore предназначен для игнорирования и не отслеживания указанных файлов.
