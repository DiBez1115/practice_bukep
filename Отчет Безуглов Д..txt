- Команда <cd <D:/Git>> - перемещает по файлам.
- Команда <git clon> - копирует репозиторий.
- Команда <git branch> - создает ветку.
- Команда <git checkout> - позволяет переключаться между ветками.
- Команда <git add -A> - добавляет отслеживание для всех новых файлов из текущей директории.
- Команда <git commit -m> - добавит в индекс все файлы, в которых были изменения.
- Команда <git merge> - выполняет слияние веток.
- Команда <git pull> - используется для извлечения и загрузки содержимого из удаленного репозитория.
- Команда <git push> - используется для выгрузки содержимого локального репозитория в удаленный репозиторий.
Файл .gitignore – это обычный текстовый файл, который содержит список всех указанных файлов и папок проекта,
которые Git должен игнорировать и не отслеживать. Для того, чтобы иметь возможность просмотреть все файлы, 
включая скрытые, используется флаг -a с командой ls.
