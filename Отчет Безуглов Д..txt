<cd D:/Git> - переместился в папку, для копирования в нее удаленного репозитория.
<git clon https://github.com/DiBez1115/practice_bukep> - копировал удаленный репозиторий.
<git branch DiBez1115> - создал ветку DiBez1115.
<git checkout DiBez1115> - переключился на ветку DiBez1115.
<git add -A> - добавляет отслеживание для всех новых файлов из текущей директории.
<git commit -m "New Laba 2"> - добавит в индекс все файлы, в которых были изменения.
<git checkout main> - переключился на ветку main.
<git merge DiBez1115> - выполняет слияние ветки main с DiBez1115.
<git pull> - использовался для извлечения и загрузки содержимого из удаленного репозитория.
<git push> - использовался для выгрузки содержимого локального репозитория в удаленный репозиторий.
Файл .gitignore – это обычный текстовый файл, который содержит список всех указанных файлов и папок проекта,
которые Git должен игнорировать и не отслеживать. Для того, чтобы иметь возможность просмотреть все файлы, 
включая скрытые, используется флаг -a с командой ls.
