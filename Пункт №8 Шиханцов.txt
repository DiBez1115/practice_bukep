Для выполнения задания использовались команды:
cd (Для перехода по папкам.)
git clone (Для клонирования репозитория.)
git branch (Для создания ветки.)
git checkout (для перехода по веткам.)
git merge (Для слияние веток.)
git add -A (Для внесения изменений в индекс.)
git commit -m "" (Сохраняет изменения индексов в историю.)
git pull (Для сравнения и внесения изменений в локальный репозиторий если они необходимы.)
git push (Для синхронизации локального и удалённого репозитария.)
Файл .gitignore предназначен для игнорирования и не отслеживания указанных файлов.
